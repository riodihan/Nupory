Apa itu PHP ?

PHP: Hypertext Preprocessor[3] adalah bahasa skrip yang dapat ditanamkan atau disisipkan ke dalam HTML.[4][5] PHP banyak dipakai untuk memprogram situs web dinamis. PHP dapat digunakan untuk membangun sebuah CMS.

Kelebihan Java.

  1. Memiliki komitas yang besar
  2. Mudah dipelajari
  3. Pengembangan ceoat
  4. Ringkas
  5. Maintenance mudah
  6. Perkembangan pesat

Kekurangan PHP.

  1. Banyak kompetisi
  2. Mudah dibajak
  3. Terkesan kurang prestigious
  4. Tidak adanya type data pada PHP
